## 超阈限空间机制复现  版本：UE5.7  

# 操作  
第一次按下左键选中物体，再次按左键放下，需要是提前规定好的物体(名字里带movable的)  

# 实现思路  
选中物体时，记录选中时物体到摄像头的距离(初始距离)，物体的World Scale(初始比例)，物体的World Location(初始位置)，物体的World Rotator(初始旋转体)，摄像头的World Rotator(摄像头初始旋转体)。  
初始距离与它背后第一个碰撞体到摄像头的距离相除得到初始比例，摄像头移动过程中保持该比例不变，此时物体到摄像头的距离除以初始距离乘初始比例则为此时物体的比例，如果没检测到物体后面有碰撞体则保持物体比例和到摄像头距离不变物体Rotator只需要保持与摄像头的相对量不变。  
采用射线检测选中物体，检测位置与物体位置存在一个偏移量，在计算时需要将物体到摄像头距离的增加量(比例)乘该偏移量，再用旋转体变化量旋转，结果加到物体新的位置上
